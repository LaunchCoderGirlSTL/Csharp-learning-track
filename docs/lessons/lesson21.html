Lesson 21: Where to Go from Here¶

This is the final class for the CoderGirl C# track. You have learned the foundation of C# and how to build Windows applications. This knowledge, while not exhaustive, has taught you to learn how to program and build real applications. This can carry you forward into any entry level programming position, even those not in C#.

If you are moving on to the project track you will continue your education with the following topics:

  • How to design and plan a more complex application
  • What is agile development
  • How to store data on a server
  • Build web sites and applications
  • How to access relational data

Prep Work¶

  • Watch this series wrap-up from the Windows 10 Development for Absolute Beginners course

Choose any one of these topics to learn more. Your coding assignment will be based on the topic you choose.

  • Watch this lesson on Connected Experiences from A Developer’s Guide to Windows 10 on how to take advantage of the user’s logged in Microsoft account
  • Watch this lesson on Background Execution from A Developer’s Guide to Windows 10 on how to keep your application running in the background after a user shuts it down
  • Watch this lesson on Databases from A Developer’s Guide to Windows 10 on how store your data in a local database. This is a more advanced topic
  • Read this article on Ink to add pen support to your application
  • Watch this lesson on Localization from A Developer’s Guide to Windows 10 on how to make your application multi-lingual
  • Read the official Microsoft documentation on how to add Maps to your application
  • Watch this video on Template Studio to learn about a tool that helps you create an application from scratch

Learning Objectives¶

  • This exercise is to prepare you to continue your education after class.

Assignments¶

The Prep Work for the final lesson is only a starting point to a topic. You will need to search for more information, coding samples and tutorials on you own.

Choose any one of the following assignments based on the Prep Work you completed above.

  • Connected Experience: Create an application that saves some data to roaming data storage. Test your application on another Windows 10 machine logged into the same account and make sure the data you save can be loaded from the other machine. Upload your program to a new repository called CG 23-1
  • Background Execution: Create an application that displays the current time. Add a Live Tile to the application that also shows the current time. Have the application continue to run in the background after it is closed so that it can continue update the time on the Live Tile. Upload your program to a new repository called CG 23-1
  • Databases: Create an application that acts as a to do list. Store that data in an SQLite database. Upload your program to a new repository called CG 23-1
  • Ink: Create an application that loads an image and allows the user to trace over it. Allow the user to see there tracing without the image also shown. Upload your program to a new repository called CG 23-1
  • Localization: Create an application that displays a paragraph of text in a language the user chooses from a list. Upload your program to a new repository called CG 23-1
  • Maps: Create an application that displays the current location of the user. Allow the user to put in a destination and tell them how long it will take to get there by car. Upload your program to a new repository called CG 23-1
  • Template Studio: Create an application using Template Studio that showcases at least one feature of the toolkit. You can decide what the application does. Upload your program to a new repository called CG 23-1

Also complete the following presentation assignment.

  • Choose any one of the following topics to research on your own to understand what it is and how it works. The possible topics include: Artificial Intelligence, Azure Cloud Services, Bots, Games, Internet of Things, Mixed Reality, Mobile or Office (https://developer.microsoft.com/en-us/office)
  • Create a PowerPoint presentation or Word document on your research topic to explain to the class what it is about and how to develop for it. You may be asked to give a 5 – 10-minute presentation to the class. Upload your document to a new repository called CG 23-2

Resources¶

  • A Developer’s Guide to Windows 10 – A series produced by Microsoft to introduce experienced developers to Windows 10. Downloadable content is available on GitHub
  • Hands on Workshop for Windows 10 Developer Training - This workshop is based on the full Windows Dev Camp set of materials, which is in turn based on the “Developer’s Guide to Windows 10” which was produced as a set of on demand video training on both Channel 9 and Microsoft Virtual Academy
  • Design and Code UWP Apps – Microsoft documentation for designing Windows 10 apps
  • Windows Universal Samples – A large collection of small sample applications that demonstrate the features of UWP applications. This is a good resource to use to understand how to implement something in your own application
  • Windows 10 Universal Platform Tutorials – Several tutorials to build small sample applications
  • Developer University – Bob’s other video-based training that is not on Microsoft Virtual Academy
  • Syncfusion Succinctly Series – A huge list of free eBooks aimed at helping you learn new things quickly. Be sure to check out UWP, More UWP, and Advanced UWP

Continuing Education¶

  • The Morning Brew – A daily update of important .NET related topics from around the web
  • Code Project - A code and tutorial sharing site
  • O’Reilly Safari Books – Thousands of online books, videos and instructor led seminars
  • Pluralsight – Professional quality video-based training
  • Programming with Mosh – A popular C# teacher’s training site
  • Food Wheels – A complete sample application with video training available on Safari Books
  • Van Arsdel Inventory – A complete line of business sample application designed by Microsoft
  • Customer Orders – Another complete line of business sample application designed by Microsoft
  • Lunch Scheduler - This app highlights integration with services like Facebook and Microsoft Graph for authentication, user operations, and friend’s discovery, services like Yelp and Bing maps for restaurant recommendations and elements of the Fluent Design System in a UWP app including acrylic, reveal, and connected animations
  • Coloring Book - UWP sample app demonstrating Inking, Surface Dial support, and printing
