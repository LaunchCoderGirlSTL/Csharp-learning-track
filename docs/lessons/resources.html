Resources Index¶

The following is a list of all material references within this course.

Books¶

  • .NET Core Succinctly - Guides you through the fundamentals of .NET Core.
  • C# Succinctly - You will quickly learn the syntax you need to build your own C# applications.
  • Clean Code and Clean Coder – Two highly praised books from Bob Martin on how to write better code. Also available in an entertaining, somewhat bizarre, video series.
  • Code Complete – Another book on writing clean code. Also, highly regarded as one of the best.
  • GitHub Succinctly – A free eBook.
  • Object Technology: A Manager’s Guide – A great introduction into object-oriented design and technology from a very non-technical perspective.
  • UWP Succinctly - The first part of a series, guides readers towards developing their own UWP applications.
  • UWP Succinctly (More) - The second part of a series, guides readers towards developing their own UWP applications.
  • UWP Succinctly (Advanced) - The third part of a series, guides readers towards developing their own UWP applications.

Documentation¶

  • .NET Application Architecture – A deep dive into building professional grade applications with .NET.
  • Microsoft Documentation - Official documentation for Microsoft technologies, including C#.
  • Stack Overflow - The best place on the internet to ask and answer questions.
  • Universal Windows Platform – Official documents for the Universal Windows Platform.
  • Windows Development Center – Microsoft’s home for all Windows development.

Libraries¶

  • Command Line Parser Library - The Command Line Parser Library offers CLR applications a clean and concise API for manipulating command line arguments and related tasks, such as defining switches, options and verb commands.
  • Humanizer - Humanizer meets all your .NET needs for manipulating and displaying strings, enums, dates, times, timespans, numbers and quantities.
  • Json.NET – A very popular NuGet library for working with JSON formatted data.
  • Monkey Cache - A library to keep a local copy of cloud data on your computer.
  • MVVM Helpers - Collection of MVVM helper classes for any application.
  • Prism – A powerful MVVM framework.
  • RestSharp – A NuGet library that makes it easier to work with REST based APIs.
  • TV Helpers - This project consists of several helper components, samples, and sample app to help you get started building applications for media, TV, and Xbox on the Universal Windows Platform for both C# and JavaScript developers.

Sample Apps¶

  • BuildCast Video Player UWP Sample - BuildCast is an end-to-end sample built to showcase the Microsoft Fluent Design System as well as other capabilities of the Universal Windows Platform.
  • Coloring Book - UWP sample app demonstrating Inking, Surface Dial support, and printing.
  • Customer Orders – Another complete line of business sample application designed by Microsoft.
  • Food Wheels – A complete sample application with video training available on Safari Books.
  • Lunch Scheduler - This app highlights integration with services like Facebook and Microsoft Graph for authentication, user operations, and friend’s discovery, services like Yelp and Bing maps for restaurant recommendations and elements of the Fluent Design System in a UWP app including acrylic, reveal, and connected animations.
  • Van Arsdel Inventory – A complete line of business sample application designed by Microsoft.
  • Windows Universal Samples – A large collection of small sample applications that demonstrate the features of UWP applications. This is a good resource to use to understand how to implement something in your own application.

Tools¶

  • JSON to C# - An online converter that will generate C# classes from JSON data.
  • Paint.net – A free application to help you edit images. It is also available in the Microsoft store for a small fee to help support the developer.
  • Rapid XAML Toolkit: Build your user interface for you based on your classes.
  • Universal Tile Maker - This script creates all the 43 PNG images that are used for Tile assets in a Universal Windows 10 application. It is optimized to generate transparent tiles which will use the system’s accent color as default and that look consistent with Microsoft’s own apps.
  • Windows Template Studio - Windows Template Studio quickly builds a UWP app, using a wizard-based UI to turn your needs into a foundation of Windows 10 patterns and best practices.

Training¶

  • A Developer’s Guide to Windows 10 – A series produced by Microsoft to introduce experienced developers to Windows 10. Downloadable content is available on GitHub.
  • C# Fundamentals for Absolute Beginners - Want to learn a different programming language? C# is a great place to start. Step through 24 practical and easy-to-understand C# training episodes, with our talented friend Bob Tabor, from Developer University, as he teaches you the fundamentals of C# programming.
  • Channel 9 - Free videos produced by Microsoft on a variety of topics and events.
  • dev up – A very affordable annual developer conference here in St. Louis that covers a lot of topics on .NET.
  • Developer University – Bob’s other video-based training that is not on Microsoft Virtual Academy.
  • Hands on Workshop for Windows 10 Developer Training - This workshop is based on the full Windows Dev Camp set of materials, which is in turn based on the “Developer’s Guide to Windows 10” which was produced as a set of on demand video training on both Channel 9 and Microsoft Virtual Academy.
  • Microsoft Virtual Academy - Free courses for learning Microsoft technology.
  • O’Reilly Safari Books – Thousands of online books, videos and instructor led seminars.
  • Pluralsight – Professional quality video-based training.
  • Programming with Mosh – A popular C# teacher’s training site.
  • Saint Louis .NET User Group – There is a local, free monthly meetup of .NET professionals for learning new things and networking with peers.
  • Windows 10 Development for Absolute Beginners - If you’d like to create Windows 10 apps on the Universal Windows Platform (UWP) and you have some experience with C#, these sessions are for you. Follow along with the coding exercises, as you learn key techniques and concepts.
