# Resources Index

The following is a list of all material references within this course.

## Books
* [.NET Core Succinctly](https://www.syncfusion.com/ebooks/net_core_succinctly) - Guides you through the fundamentals of .NET Core.
* [C# Succinctly](https://www.syncfusion.com/ebooks/csharp) - You will quickly learn the syntax you need to build your own C# applications.
* [Clean Code](https://www.amazon.com/Clean-Code-Handbook-Software-Craftsmanship/dp/0132350882/) and [Clean Coder](https://www.amazon.com/Clean-Coder-Conduct-Professional-Programmers/dp/0137081073/) – Two highly praised books from Bob Martin on how to write better code.  Also available in an entertaining, somewhat bizarre, [video series](https://cleancoders.com/).
* [Code Complete](https://www.amazon.com/Code-Complete-Practical-Handbook-Construction/dp/0735619670/) – Another book on writing clean code.  Also, highly regarded as one of the best.
* [GitHub Succinctly](https://www.syncfusion.com/ebooks/github_succinctly) – A free eBook.
* [Object Technology: A Manager’s Guide](https://www.amazon.com/Object-Technology-Managers-David-Taylor/dp/0201309947) – A great introduction into object-oriented design and technology from a very non-technical perspective.
* [UWP Succinctly](https://www.syncfusion.com/ebooks/uwp_succinctly) - The first part of a series, guides readers towards developing their own UWP applications.
* [UWP Succinctly (More)](https://www.syncfusion.com/ebooks/more_uwp_succinctly) - The second part of a series, guides readers towards developing their own UWP applications.
* [UWP Succinctly (Advanced)](https://www.syncfusion.com/resources/techportal/whitepapers) - The third part of a series, guides readers towards developing their own UWP applications.

## Documentation
* [.NET Application Architecture](https://www.microsoft.com/net/learn/architecture) – A deep dive into building professional grade applications with .NET.
* [Microsoft Documentation](https://docs.microsoft.com/en-us/) - Official documentation for Microsoft technologies, including C#.
* [Stack Overflow](https://stackoverflow.com/) - The best place on the internet to ask and answer questions.
* [Universal Windows Platform](https://docs.microsoft.com/en-us/windows/uwp/index) – Official documents for the Universal Windows Platform.
* [Windows Development Center](https://developer.microsoft.com/en-us/windows) – Microsoft’s home for all Windows development.

## Libraries
* [Command Line Parser Library](https://github.com/commandlineparser/commandline) - The Command Line Parser Library offers CLR applications a clean and concise API for manipulating command line arguments and related tasks, such as defining switches, options and verb commands.
* [Humanizer](https://github.com/Humanizr/Humanizer) - Humanizer meets all your .NET needs for manipulating and displaying strings, enums, dates, times, timespans, numbers and quantities.
* [Json.NET](https://www.newtonsoft.com/json) – A very popular NuGet library for working with JSON formatted data.
* [Monkey Cache](https://montemagno.com/data-caching-made-simple-with-monkey-cache/) - A library to keep a local copy of cloud data on your computer.
* [MVVM Helpers](https://github.com/jamesmontemagno/mvvm-helpers) - Collection of MVVM helper classes for any application.
* [Prism](https://github.com/PrismLibrary/Prism) – A powerful MVVM framework.
* [RestSharp](http://restsharp.org/) – A NuGet library that makes it easier to work with REST based APIs.
* [TV Helpers](https://github.com/Microsoft/TVHelpers) - This project consists of several helper components, samples, and sample app to help you get started building applications for media, TV, and Xbox on the Universal Windows Platform for both C# and JavaScript developers. 

## Sample Apps
* [BuildCast Video Player UWP Sample](https://github.com/Microsoft/BuildCast) - BuildCast is an end-to-end sample built to showcase the Microsoft Fluent Design System as well as other capabilities of the Universal Windows Platform.
* [Coloring Book](https://github.com/Microsoft/Windows-appsample-coloringbook) - UWP sample app demonstrating Inking, Surface Dial support, and printing.
* [Customer Orders](https://github.com/Microsoft/Windows-appsample-customers-orders-database) – Another complete line of business sample application designed by Microsoft.
* [Food Wheels](https://github.com/colinizer/foodwheels) – A complete sample application with video training available on Safari Books.
* [Lunch Scheduler](https://github.com/Microsoft/Windows-appsample-lunch-scheduler) - This app highlights integration with services like Facebook and Microsoft Graph for authentication, user operations, and friend’s discovery, services like Yelp and Bing maps for restaurant recommendations and elements of the Fluent Design System in a UWP app including acrylic, reveal, and connected animations.
* [Van Arsdel Inventory](https://github.com/Microsoft/InventorySample) – A complete line of business sample application designed by Microsoft.
* [Windows Universal Samples](https://github.com/Microsoft/Windows-universal-samples) – A large collection of small sample applications that demonstrate the features of UWP applications.  This is a good resource to use to understand how to implement something in your own application.

## Tools
* [JSON to C#](http://json2csharp.com/) - An online converter that will generate C# classes from JSON data.
* [Paint.net](https://www.getpaint.net/) – A free application to help you edit images.  It is also available in the Microsoft store for a small fee to help support the developer.
* [Rapid XAML Toolkit](https://github.com/Microsoft/Rapid-XAML-Toolkit): Build your user interface for you based on your classes.
* [Universal Tile Maker](https://github.com/Aftnet/UniversalTileMaker) - This script creates all the 43 PNG images that are used for Tile assets in a Universal Windows 10 application. It is optimized to generate transparent tiles which will use the system's accent color as default and that look consistent with Microsoft's own apps.
* [Windows Template Studio](https://github.com/microsoft/windowsTemplateStudio) - Windows Template Studio quickly builds a UWP app, using a wizard-based UI to turn your needs into a foundation of Windows 10 patterns and best practices.

## Training
* [A Developer’s Guide to Windows 10](https://mva.microsoft.com/en-US/training-courses/a-developers-guide-to-windows-10-12618) – A series produced by Microsoft to introduce experienced developers to Windows 10.  Downloadable content is available on [GitHub](https://github.com/Windows-Readiness/WinDevCamp).
* [C# Fundamentals for Absolute Beginners](https://mva.microsoft.com/en-US/training-courses/c-fundamentals-for-absolute-beginners-16169) - Want to learn a different programming language? C# is a great place to start. Step through 24 practical and easy-to-understand C# training episodes, with our talented friend Bob Tabor, from Developer University, as he teaches you the fundamentals of C# programming.
* [Channel 9](https://channel9.msdn.com/) - Free videos produced by Microsoft on a variety of topics and events.
* [dev up](https://devupconf.org/) – A very affordable annual developer conference here in St. Louis that covers a lot of topics on .NET.
* [Developer University](https://devu.com/) – Bob’s other video-based training that is not on Microsoft Virtual Academy.
* [Hands on Workshop for Windows 10 Developer Training](https://github.com/Windows-Readiness/WinDevWorkshop) - This workshop is based on the full Windows Dev Camp set of materials, which is in turn based on the "Developer's Guide to Windows 10" which was produced as a set of on demand video training on both Channel 9 and Microsoft Virtual Academy.
* [Microsoft Virtual Academy](https://mva.microsoft.com/) - Free courses for learning Microsoft technology.
* [O’Reilly Safari Books](https://www.safaribooksonline.com/) – Thousands of online books, videos and instructor led seminars.
* [Pluralsight](https://www.pluralsight.com/) – Professional quality video-based training.
* [Programming with Mosh](https://programmingwithmosh.com/) – A popular C# teacher’s training site.
* [Saint Louis .NET User Group](https://www.meetup.com/St-Louis-NET-User-Group/) – There is a local, free monthly meetup of .NET professionals for learning new things and networking with peers.
* [Windows 10 Development for Absolute Beginners](https://mva.microsoft.com/en-US/training-courses/windows-10-development-for-absolute-beginners-14541) - If you'd like to create Windows 10 apps on the Universal Windows Platform (UWP) and you have some experience with C#, these sessions are for you. Follow along with the coding exercises, as you learn key techniques and concepts.
