Lesson 19: File Handling & Monetization¶

Prep Work¶

  • Watch and code along with this lesson introducing the album cover project from the Windows 10 Development for Absolute Beginners course. Make sure you code along with Bob during this series. See Assignment 19-1
  • Watch and code along with this lesson for setting up and working with files and system folders in the album cover project
  • Watch and code along with this lesson for layout, data binding and game setup in the album cover project
  • Watch and code along with this lesson on employing game logic in the album cover project
  • Watch and code along with this lesson on evaluating user input and tracking progress in the album cover project
  • Watch and code along with this lesson to enable the play again feature in the album cover project
  • Watch and but do NOT code along with this lesson on monetizing the album cover project with advertisements. This Ad SDK has been deprecated. The next lessons will direct you to the current version
  • Install the Microsoft Advertising SDK by completing the “Install via NuGet” and “Add a reference to the Microsoft Advertising SDK” sections of this article
  • Read the instructions for adding advertisements to you app. Use this control to add an advertisement to the album cover project like what Bob has done
  • Watch and code along with this lesson to add in-app purchases for advertising removal to the album cover project
  • Watch this video from Channel 9 that goes more in depth on file management

Learning Objectives¶

  • Work with special file folders like documents, music, pictures and videos
  • Read metadata from files
  • Use story boards to animate controls
  • Monetize your application with advertisements
  • Monetize your application with in-app purchases

Assignments¶

  • Take the Setup and Working with Files and System Folders Assessment
  • Take the Layout, Data Binding, and Game Setup Assessment
  • Take the Employing Game Logic Assessment
  • Take the Evaluating User Input and Tracking Progress Assessment
  • Take the Enabling the Play Again Feature Assessment
  • Take the In-App Purchase for Ad Removal Assessment
  • Code along with Bob to create the Album Cover Match Game application. Again, please be sure to comment your code. Upload your program to a new repository called CG 19-1
  • Update the application to start playing the song in the middle instead of the beginning. Upload your program to a new repository called CG 19-2
  • Create a new program with a large multiline text box and save button. When the user’s clicks the save, it will save the contents to a file. You do not need to let the user choose the file location. When the user restarts the application, it should automatically load the contents of this file. Upload your program to a new repository called CG 19-3
  • Update the previous program by removing the save button and having the file save changes automatically. Upload your program to a new repository called CG 19-4

  • Enable In-App Product Purchases – Microsoft documentations on monetizing your application through in-app purchases
  • Work with Files – Microsoft documentation on reading and writing files
